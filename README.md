# 📚 The Psychology of Money - Landing Page

A simple and modern landing page inspired by the bestselling book **The Psychology of Money** by Morgan Housel.

## ✨ Features

* Clean and attractive design
* Responsive layout
* Book showcase section
* Key insights from the book
* Call-to-action button
* Easy navigation

## 🛠️ Built With

* HTML5
* CSS3

## 🎯 Purpose

This project was created to practice HTML and CSS by designing a visually appealing landing page for an e-book.

## 📖 About the Book

*The Psychology of Money* explores how people think about money and explains why financial success is often driven more by behavior than knowledge.

## 🚀 What I Learned

* Styling with CSS
* Responsive design
* Creating attractive user interfaces
